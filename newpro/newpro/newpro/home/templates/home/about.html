Automatic Timetable Generator

The Automatic Timetable Generator is a web-based application developed using Django. It automatically creates structured and conflict-free academic timetables for institutions. The system takes into account courses, teachers, classrooms, and time constraints to generate optimized schedules without manual effort. This project helps institutions save time, reduce errors, and provide better planning for students and faculty.

Advantages

- Saves significant time compared to manual scheduling.
- Reduces conflicts between courses and teachers.
- Provides clear and structured timetable for students and staff.
- Can handle multiple constraints efficiently.
- Improves planning and organization in educational institutions.

Disadvantages

- Requires correct and complete input data for accurate results.
- May not handle unexpected changes dynamically.
- Initial setup may be time-consuming.
- Limited customization without modifying the code.
- Depends on the availability of a stable web server and database.
